# Java Bill Generator

## Description
Este proyecto es lo mas completo que he hecho hasta la fecha (16/07/2026), se trata de una aplicación de generador de factura sencilla pero muy util. Desarrollada en el curso del SENA usando Java Swing

## How  does it work?
Funciona de una manera muy sencilla, sirve para generar una factura de compra para tres productos, aqui el usuario puede ingresar lo siguiente:

- Nombre del cliente
- Nombre de los 3 productos
-  Valor unitario de cada producto
- Cantidad de cada producto

El sistema calcula automaticamente lo siguiente al presionar el boton "CALCULAR"

- Total de cada producto
- Total general de la factura
- Un mensaje final con el nombre del cliente y el valor total de la compra
